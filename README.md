# sml-NJ-portable
This is for portable install of sml/NJ on windows 

Unzip the file to a location. 
Open "Edit environment variables for your account"
Hit New.. on upper section and add:
  Variable Name: SMLNJ_HOME
  Variable Value: Example: D:\SMLNJ
Hit Edit.. on Path
  Add: Example D:\SMLNJ\bin

Now, SML/NJ should work just fine.
